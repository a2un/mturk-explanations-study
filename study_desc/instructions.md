## Study Title
**Evaluate Explanations of Computer Code**

## Description
In this study, you will be shown a short snippet of computer code and an explanation written by a student for a specific line of that code. Your task is to evaluate the explanation based on how accurately and clearly it describes what the highlighted line of code does.

You are **not** required to write code or provide an explanation yourself. Instead, you will judge whether the explanation correctly reflects the behavior and purpose of the given line of code.

## What You Will Do
- Read a short code snippet  
- Focus on a single highlighted line of code  
- Read a student’s explanation of that line  
- Answer a small number of questions assessing the explanation’s correctness and quality  

## Requirements
- Basic familiarity with introductory programming concepts (e.g., variables, loops, conditionals)  
- Careful reading and attention to detail  

## Time Commitment
Approximately **2–4 minutes per task**

## Compensation
Payment is provided for each completed task and is proportional to the time required.

## Data Use and Privacy
Your responses will be used for academic research on computer science education. No personally identifying information will be collected.

## Note
There are no trick questions. Please answer honestly based on your understanding of the code and the explanation provided.
